

 Gym Deeb Dashboard



A responsive front-end web application designed to manage gym memberships, track financial records, and monitor subscription statuses.

## 🌟 Features

* **Member Management**: Add, edit, delete, and renew gym memberships with automatic expiration tracking.


* **Dashboard Statistics**: Instantly view total members, active subscriptions, expired members, and members expiring within 7 days.


* **Finance Panel**: A secure treasury section protected by an owner login to view total revenue and individual member payments.


* **Search Functionality**: Quickly filter and find members using their name or phone number.


* **Data Export**: Export the complete member list and their current statuses directly to a CSV file.


* **Dark Mode**: A toggleable dark and light theme that saves your preference for future sessions.


* **Modern UI**: Features a smooth animated gradient background, glassmorphism panel effects, and interactive hover animations.



## 💻 Technologies Used

* **HTML5**

* **CSS3**: Custom styling utilizing CSS variables and keyframe animations.


* **JavaScript (Vanilla)**: Handles application logic and utilizes the browser's `localStorage` for data persistence.


* **Bootstrap 5.3.0**: Utilized for the responsive layout, modals, and pre-built UI components.


* **Bootstrap Icons 1.11.3**


## 🚀 Setup and Usage

* **No Installation Required**: Because the application uses the browser's `localStorage` to save all member data, no backend database setup is required.


* **How to Run**: Simply open the HTML document in any modern web browser to launch the dashboard.


* **Admin Credentials**: To unlock and access the secure Finance Panel, click the "Finance Panel" button and log in using the following default credentials:
* **Email**: `admin@gym.com`

* **Password**: `123456`
